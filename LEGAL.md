**Project:** FrameDrawing - Image Display System

## Description

This is a Flask-based web application designed to display images on multiple monitors simultaneously. It features a WebSocket-driven architecture for real-time image updates, barcode scanner integration, and REST API endpoints for external control. The system includes a central control panel and individual display screens configurable via JSON.

The tool was created as a **general-purpose image display system**, adaptable to any use case requiring remote image control—manufacturing, digital signage, presentations, galleries, etc.
